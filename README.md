# docker-year

Just a docker CLI plugin.

## Installing


```
(MoeLove) ➜  ~ mkdir -p ~/.docker/cli-plugins
(MoeLove) ➜  ~ wget -qO ~/.docker/cli-plugins/docker-year https://raw.githubusercontent.com/tao12345666333/docker-year/master/docker-year.sh 
(MoeLove) ➜  ~ docker year
Happy New Year 2020!
```
